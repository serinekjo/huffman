SUJET : Compresseur de Huffman de fichiers 

Compilation: gcc -o huffman huff.c -Wall

Exécution: ./huffman <fichier> 

Note: 

	- Le programme permet seulement de compresser le fichier texte donné en argument: absence de l'option de décompression

	- Le principe est de compresser le fichier en utilisant une représentation binaire des caractères basée sur leur fréquence d'apparition.
